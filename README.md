# AI Task Board Frontend Skeleton

Production-ready MVP skeleton for an AI Task Board frontend built with React, TypeScript, Vite, React Router, Zustand, TanStack Query, Supabase client setup, Tailwind CSS, shadcn-style UI primitives, React Hook Form and Zod.

## Run locally

1. Install dependencies:
   - `npm install`
2. Configure env:
   - copy `.env.example` to `.env`
3. Start dev server:
   - `npm run dev`

## Available scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run typecheck`
- `npm run lint`
